# verso-test

This repository contains a simple test for [verso](https://github.com/leanprover/verso).

## How to run it

Install [lean4](https://lean-lang.org/), clone this repository and then run
```
lake exe verso-test
```

This creates `index.html` und `docs/html-single`.

## See the result

That verso generated can be found [here](https://runbikeswim.github.io/verso-test/html-single/index.html) on GitHub pages.
